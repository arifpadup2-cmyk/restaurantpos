Auto-Recovery

60-Second Sync Cycle

Every 60 seconds, LOCAL server pulls from CLOUD:

  • 📋 Menu items
  • 🏷️ Categories
  • 👥 Staff & Cashiers
  • ⚙️ Settings
  • 🪑 Tables layout
  • 🖨️ Printers
  • 🎯 Modifiers

⚠️ Auto-Failover Logic

Apps check LOCAL server every 5 seconds via health check. After 3 consecutive failures (15 seconds), automatically switch to CLOUD mode. On recovery, switch back to LOCAL.
