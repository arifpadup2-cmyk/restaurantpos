📋 Step-by-Step Installation

Step 1 — Server PC Setup (Run once, on the main restaurant computer)

  1. Copy the server folder to the restaurant's server PC
  2. Right-click INSTALL-SERVER.bat → Run as Administrator
  3. When prompted, enter the Restaurant ID and License Key from the credentials below
  4. Script auto-installs Node.js, PostgreSQL, configures everything and starts the server as a Windows service
  5. It saves a SETUP-SUMMARY.txt file with all credentials and the server IP address

Step 2 — POS Terminal (on each cashier/waiter Windows PC)

  1. Download and run Restaurant POS Setup X.X.X.exe
  2. On first launch, a setup screen appears — enter Server IP, Restaurant ID, License Key
  3. Click Connect & Save — POS downloads its database credentials from the server and connects
  4. Enter staff PIN to login and start using the POS

Step 3 — Web Apps (Waiter App, Kitchen Display, Back Office)

  1. On any device connected to the restaurant's LAN, open a browser
  2. Back Office: http://<server-ip>:3001/backoffice/ — login with Back Office credentials
  3. Waiter App: http://<server-ip>:3001/waiter-app/ — tap the screen bookmark/install as PWA
  4. Kitchen Display: http://<server-ip>:3001/kds/ — open on kitchen screen, set to kiosk mode

Step 4 — Back Office First-Time Setup

  1. Login to Back Office with the provided username and password
  2. Go to Settings → enter Restaurant Name, currency, tax rate
  3. Go to Menu → add categories and menu items
  4. Go to Staff → add cashiers and waiters with PINs
  5. Go to Tables (if dine-in) → configure table layout
  6. Open the POS, login with a cashier PIN — ready to go!

⚠️ Important Notes

  1. The server PC must always be on when the restaurant is operating — all terminals depend on it
  2. Server PC should be connected via LAN cable (not WiFi) for reliability
  3. Firewall ports that must be open: 3001 (API) and 5432 (PostgreSQL) — the install script opens these automatically
  4. Auto-backups run every 6 hours and are kept for 14 days — downloadable from Back Office → Downloads
  5. If server PC is replaced: restore from backup via Back Office → Downloads → Restore
